# Memory Card Game

A classic card-matching memory game.

## Overview

Build a memory card game using vanilla HTML, CSS, and JavaScript. Players flip cards to find matching pairs. The game tracks moves and time, encouraging players to improve their score.

## Acceptance Criteria

- 4x4 grid of face-down cards (8 pairs)
- Cards flip with smooth CSS animation on click
- Two cards can be flipped per turn
- Matching pairs stay revealed; non-matches flip back
- Move counter increments each turn
- Timer starts on first card flip
- Win screen shows when all pairs found with final stats
- New Game button shuffles and resets
